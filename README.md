# The Weather Service

Code kata using the [Weatherbit.io](https://developer.nytimes.com/), [IPapi.co](https://IPapi.co/) and [geolocation](https://developer.mozilla.org/en-US/docs/Web/API/Geolocation_API) API's.

Icons: [weather-icons](https://erikflowers.github.io/weather-icons/) & [Moon Phases](https://www.flaticon.com/packs/moon-phase-3)
